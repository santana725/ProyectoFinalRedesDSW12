Proyecto de Redes – ITCA FEPADE Zacatecoluca

Este repositorio contiene la documentación del proyecto de levantamiento, 
análisis y propuesta de mejora de la infraestructura de red del Instituto ITCA–FEPADE Zacatecoluca.
El objetivo principal es presentar de forma organizada todos los archivos, evidencias y 
avances desarrollados por el grupo durante el módulo de Redes.
